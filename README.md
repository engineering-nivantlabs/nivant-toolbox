# nivant-toolbox

Collection of 50+ AI-powered tools — essays, recipes, code explanation, image generation, and more

## Tech Stack

React, TypeScript, Vite, Tailwind CSS, Shadcn UI

## Quick Start

```bash
git clone https://github.com/engineering-nivantlabs/nivant-toolbox.git
cd nivant-toolbox
npm install
cp .env.example .env
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Environment Variables

Copy `.env.example` to `.env` and fill in your keys:

```bash
cp .env.example .env
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

Configured for Vercel — push to `main` and it auto-deploys.
